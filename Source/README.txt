Khuyến cáo:
- File Lab2_Logistic_Final.ipynb và Train_All_Data chỉ chạy nếu RAM máy 
trên 8GB. Còn RAM 8GB hoặc dưới thì nên tắt mấy chương trình chạy ngầm.

Thư mục Model:
- File Lab2_Logictic_Final.ipynb thực hiện phân tích và tiến hành
các thử nghiệm.
- File Train_All_Dataset.ipynb thực hiện huấn luyện mô hình tốt nhất
 ở các thử nghiệm với toàn bộ tạo dữ liệu và tạo ra file pipeline_LR.sav
- File pipeline_LR.sav lưu model đã huấn luyện, cần để chạy được App.

Thư mục App chứa chương trình demo vẽ số và dự đoán.
